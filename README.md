# Cake.FluentMigrator

[![License](http://img.shields.io/:license-mit-blue.svg)](http://cake-contrib.mit-license.org)

Cake.FluentMigrator is an Addin for [Cake](http://cakebuild.net/) which allows the execution of the FluentMigrator tool.
